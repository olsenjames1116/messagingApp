## Description

A messaging app to chat back and forth with friends. Sign up for an account, chat and edit your profile as you see fit.

This project highlights:

- Express/Node.js
- React
- MongoDB
- Cloudinary
- Redis
- Redux
- MVC architecture
- CRUD operations
